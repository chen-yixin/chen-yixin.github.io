来源

通信术语

  1. 信息
  2. 数据 信息的载体
  3. 信号
    1. 模拟信号
    2. 数字信号
  4. 码元 单位时间内信号波形变换次数

数据通信系统模型

信道

  1. 带宽 (Hz)
  2. 码元速率 理想状态下 (波特Baud) (奈奎斯特定理)
  3. 数据速率 (bps)
  4. 极限数据速率 (bps)

调制技术

模拟信道传输数字数据

调制方式 说明 码元种类 比特位
ASK 幅移键控 2 1
FSK 频移键控 2 1
PSK 相移键控 2 1
2DPSK 二差分幅移键控(=PSK) 2 1
4DPSK 四差分幅移键控 4 2
QPSK 正交相移键控 4 2
MQAM 正交幅度调制 M -

编码技术

数字信道传输模拟数据

  1. PCM 脉冲编码调制
    1. 采样
      • 采样频率大于模拟信号最高频率的2倍
    2. 量化
    3. 编码

数字编码

  1. 极性编码方案 需引入时钟信号
    1. 单极性码 高电平->0, 0电平->1
    2. 极性码 高电平->0, 低电平->1
    3. 双极性码 0电平->0, 高低电平->1(翻转上一次电平)
  2. 归零信道编码方案
    1. 归零码RZ 高电平->0, 低电平->1, 中间信号回归0电平
    2. 不归零码NRZ 0->不变, 1->翻转电平
    3. 双相码
  3. 应用性编码
    1. 曼彻斯特编码
      • 高->低:1, 低->高:0
      • 有自同步能力,不需要引入时钟信号
      • 常用于10M以太网
      • 编码效率50%
    2. 差分曼彻斯特编码
      • 位之间, 翻转电平:0, 不翻转电平:1
      • 有自同步能力,不需要引入时钟信号
      • 用于令牌环网
  4. 其他编码
    1. 4B/5B
      • 效率: 1.25波特/位, 80%
      • 应用: 100Base-FX, 100Base-TX, FDDI
    2. 8B/10B
      • 效率: 1.25波特/位, 80%
      • 应用: 千兆以太网
    3. 8B/6T

复用技术

  1. 空分复用SDM 不同空间内统一频率(卫星通信)
  2. 频分复用FDM
  3. 波分复用WDM 和FDM类似,光纤中
  4. 时分复用TDM 划分时间片/时隙,同步、异步(统计时分复用)
  5. 码分复用CDMA

复用标准PDH

同步时分复用技术

  1. E载波(欧洲)
    1. E1 30个话音信道(64K)+2个控制信道CH0+CH16(64K) => 2.048Mbps
    2. E2 8.488Mbps (4E1)
    3. E3 34.368Mbps (4E2)
    4. E4 139.264Mbps (4E3)
  2. T载波(美国、日本)
    1. T1 24个话音通路复合在1.544Mbps通路
    2. T2 4T1 => 6.312Mbps
    3. T3 7T2 => 44.736Mbps
    4. T4 6T3 => 274.176Mbps

通信方式

  1. 单工双工
    1. 单工通信 A->B, 例如广播
    2. 半双工通信 A<->B, 每次单向通信
    3. 全双工通信 A<=>B, 双向同时通信
  2. 同步异步
    1. 异步传输 起始位_数据_结束位
    2. 同步传输 起始位_帧

差错控制

  1. 检错、纠错
  2. 码距 一个编码系统中两个码字之间需要最少的变化位数
  3. 码距、检错、纠错
    1. 检测e个误码,要求最小码距满足: d >= e + 1
    2. 纠正t个误码,要求最小码距满足: d >= 2t + 1
  4. 检错码
    1. 构造 检错码 = 信息字段 + 校验字段
    2. 奇偶校验 信息+校验奇数个/偶数个
    3. 海明校验码
      1. m位用户数据中加入k位校验码,满足
      2. 可以检测和纠正1位错误
    4. CRC校验
      1. 生成多项式最高幂次 = CRC位数
      2. 计算过程模2除,不进位不借位
      3. 只能检错不能纠错
      4. CRC32用于以太网协议802.3
